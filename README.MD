# iCloud Photos Web Downloader

一个基于 **Playwright** 的非官方脚本，用来通过 **iCloud 网页端** 下载照片原片。

它会自动打开浏览器、进入 `icloud.com.cn`、协助进入“照片”页面，并从你在 `config.json` 中指定的照片位置继续向后逐张下载。下载完成后，脚本还会尽量保留照片时间信息，并在遇到压缩包时自动解压出其中的 `iCloud 照片` 目录内容。

> **注意**
>
> - 这是一个**非官方**项目，与 Apple 无关。
> - 仅建议用于**个人合法备份、归档或迁移自己的 iCloud 照片**。
> - 登录密码、双重验证、设备信任等步骤仍需你在网页中**手动完成**。
